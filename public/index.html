Request Header Parsing Microservice



API instructions

Visit the URL [project-url]/api/whoami in order to receive a JSON object displaying your IP address, browser software, and preferred language, which has been parsed from the request sent from your browser to the server.

Please see the example URL and expected output below.


Example usage


[project-URL]/api/whoami

Example output

{"ipaddress":"159.20.14.100",
"language":"en-US,en;q=0.5",
"software":"Mozilla/5.0 (X11; Ubuntu; Linux x86_64; rv:50.0) Gecko/20100101 Firefox/50.0"}
